Установка и настройка:
   1. Открой консоль в папке с проектом и введи для установки библиотек: `pip install -r requirements.txt`.
   2. Открой для редактирования `main.py` и вставь в переменную `TOKEN` токен бота.
   3. Открой для редактирования `camera_manager.py` и вставь в переменную `cam_url` ссылку или ip на камеру.
   4. Запусти `API.py` и `main.py`.

Запуск:
   1. В чате с ботом напиши команду `/start`.
   2. Для остановки напиши команду `/stop`.
